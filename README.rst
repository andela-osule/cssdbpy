.. image:: https://readthedocs.org/projects/ssdbpy/badge/?version=latest
------

SSDBPy is a simple SSDB client written on Cython. 5x faster standart ssdb client.


Install
-------

.. code-block:: bash

   pip install ssdbpy

or

.. code-block:: bash

   pip install https://github.com/deslum/ssdbpy
